# Azure IT Support Ticket Simulation

## Project Overview
This project simulates an entry-level IT support ticket using Microsoft Azure.  
The goal is to demonstrate how an IT technician investigates an issue, performs safe actions in Azure, documents findings, and cleans up resources to avoid unnecessary costs.

---

## Simulated Ticket
**Ticket ID:** AZ-001  
**Issue Type:** Cloud Storage Access / Configuration  
**Priority:** Medium  
**Reported By:** End User  
**Environment:** Microsoft Azure

---

## Problem Description
A user reports difficulty accessing a file stored in an Azure Storage Account.  
IT support is tasked with verifying the storage resource, reviewing configurations, making a test change, and confirming access.

---

## Tools & Technologies Used
- Microsoft Azure Portal
- Azure Resource Groups
- Azure Storage Account
- Cost Management
- Windows OS
- Documentation & screenshots

---

## Actions Taken
1. Logged into Azure Portal
2. Created a Resource Group
3. Created an Azure Storage Account
4. Uploaded a test file
5. Edited the file in Azure
6. Downloaded the updated file to confirm changes
7. Verified no unexpected costs
8. Deleted the Resource Group
